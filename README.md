# 📖 Bible-Corner

Bible-Corner is a **dummy version of a Bible website** built as a learning project.  
It demonstrates how to load and display Bible text using **JavaScript** and simple frontend techniques.  
The project uses `.txt` or `.csv` files stored locally (New Testament books) and loads them dynamically into the webpage.

---

## ✨ Features
- Load New Testament books from local `.txt` files inside the `archive` folder.
- Display book content directly in the browser using JavaScript `fetch()`.
- Simple, clean UI built with **HTML + CSS**.
- Beginner-friendly project for practicing file loading and frontend development.
